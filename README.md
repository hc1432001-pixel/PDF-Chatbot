# PDF Chatbot

A Retrieval-Augmented Generation (RAG) style PDF chatbot built using Streamlit, FAISS, and Sentence Transformers.

## Features

* Upload PDF documents
* Extract text automatically
* Semantic search using Sentence Transformers
* Fast retrieval using FAISS
* Find the most relevant sections for a query
* Deployable on Hugging Face Spaces

## Tech Stack

* Python
* Streamlit
* Sentence Transformers
* FAISS
* PyPDF
* NumPy

## Future Improvements

* LLM-powered answer generation
* Groq integration
* Chat history
* Multi-PDF support
* Source citations
* Better chunking strategies
